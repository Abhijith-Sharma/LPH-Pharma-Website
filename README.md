LPH Pharma Website

A live, production website built end-to-end for a pharmaceutical distribution company — designed, coded, and deployed using Claude Code.

About:
This project was built independently to create a clean, professional web presence for a pharma distribution business. Rather than using a website builder or hiring a developer, I used Claude Code to go from concept to a fully functional, live website — handling structure, styling, content, and basic interactivity myself.

What it includes:
Home page — company overview, mission, and vision
Contact page — contact details and inquiry flow
Responsive layout and styling
Custom JavaScript for site interactivity

Built with:
HTML
CSS
JavaScript
Claude Code — used for end-to-end development, from initial structure to final polish
